# XbeeTransceiver

# Hardware

- 2 Raspberry Pi's 2
- 2 Xbee's Series 1

# Software
[https://pypi.python.org/pypi/pyserial)

- Python 2.7
- Pyserial

This code uses the UART pins on the Raspberry Pi 2 with a baudrate of 57600
 

# Play received wav on the pi
```
omxplayer --vol "-600" <your_wav_file.wav>
```
